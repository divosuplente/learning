## Course Modules

- **[00 — Java Foundations](00-java-foundations/)** — Records, generics, pattern matching, sealed classes, collections, OOP, functional interfaces.
- **[01 — Build Tools & Project Setup](01-build-tools-and-project-setup/)** — Maven, Gradle, Spring Initializr, project structure, conventions.
- **[02 — Dependency Injection](02-dependency-injection/)** — DI as a pattern, Spring IoC container, bean lifecycle, scopes, qualifiers, AOP.
- **[03 — Spring Boot Fundamentals](03-spring-boot-fundamentals/)** — Auto-configuration, starter dependencies, application properties, profiles, actuator.
- **[04 — Repository Pattern](04-repository-pattern/)** — Spring Data JPA, repositories, custom queries, transactions, domain modeling.
- **[05 — Service Oriented Architecture](05-service-oriented-architecture/)** — Service-oriented architecture, layering, DTOs, controllers, business logic separation.
- **[06 — Kafka](06-kafka/)** — Producers, consumers, topics, partitions, event-driven architecture, Kafka Streams.
- **[07 — GraphQL](07-graphql/)** — Schema-first design, queries, mutations, subscriptions, Spring for GraphQL.
- **[08 — Reactor Pattern](08-reactor-pattern/)** — Reactive Streams spec, Project Reactor, Mono/Flux, backpressure, WebFlux.
- **[09 — TDD](09-tdd/)** — JUnit 5, Mockito, test doubles, Spring Boot Test, integration testing, coverage.
- **[10 — Capstone Project](10-capstone-project/)** — Full-stack order management system using all technologies. Sealed classes, virtual threads, Kotlin preview.
- **[11 — Migrating Java to Kotlin](11-migrating-java-to-kotlin/)** — Data classes, coroutines, null safety, interoperability, migration strategy.
