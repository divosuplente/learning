# Java • Spring Boot • Kafka • GraphQL • Reactor • TDD • Kotlin

## A Complete Course for Beginners

This course teaches you everything from zero to building a production-grade backend service using Java, Spring Boot, Kafka, GraphQL, the Reactor pattern, Test-Driven Development, and Kotlin migration.

### Prerequisites

- A computer running macOS, Linux, or Windows
- willingness to read, type code, and experiment
- No prior programming experience required

### How to Use This Course

1. **Read each module in order.** Each builds on the previous one.
2. **Type every code example yourself.** Don't copy-paste. Muscle memory matters.
3. **Do every exercise.** They are designed to reinforce concepts.
4. **Refer back.** Later modules reference earlier ones — that's normal.
5. **Use the index files.** Each module starts with a "What you'll learn" section.
