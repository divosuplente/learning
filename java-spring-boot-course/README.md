# Backend Engineering Course

## Java • Spring Boot • Kafka • GraphQL • Reactor • TDD • Kotlin

A comprehensive course for active developers who need to learn Java and the surrounding backend ecosystem. Assumes you already know how to program — we skip "what is a variable" and go straight into Java idioms, framework patterns, and production architecture.

### Prerequisites

- Programming experience in any language (Python, JS, C#, Go, etc.)
- Basic understanding of HTTP, REST, and databases
- A computer running macOS, Linux, or Windows
- No prior Java experience required — but we move fast

### How to Use This Course

1. **Read each module in order.** Each builds on the previous one.
2. **Type every code example yourself.** Don't copy-paste. Muscle memory matters.
3. **Use the table of contents.** Each module has a collapsible TOC at the top for navigation.
4. **Expand submodules.** Sections marked with ▶ can be expanded for deeper coverage.
5. **Refer back.** Later modules reference earlier ones — that's normal.

### Recommended Reading Order for Review

If you already know some Java and just need the architecture pieces:

`02 → 03 → 04 → 05 → 08 → 06 → 07 → 09 → 10 → 11`
